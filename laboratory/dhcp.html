DHCP Role and Joining Workstations in Domain

Add roles: DHCP Server
Scopes:
- number of male students per yr and section is the host requirement with the ip block 10.0.0.0 /8.
- number of female students per yr and section is the host requirement with the ip block 172.16.0.0/16
- Server: Last Usable IP add
- Exclude: First & Last (10 usable IP Addresses)
- Lease Time: 16 hours
- Router: Server IP Address
- DNS: Server IP Address (8.8.8.8)

1. Create client OS, 1 Windows XP (LASTNAME-XP), and 1 Windows 7(LASTNAME-WIN7)
2. Connect the two (2) client OS to your Windows Server Domain USING DHCP.
- WINDOW XP IP ADDRESS MUST BELONG TO CLASS A NETWORK
- WINDOW 7 IP ADDRESS MUST BELONG TO CLASS B NETWORK
3. LOGIN 5 student accounts per section per OS
